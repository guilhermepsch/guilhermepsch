Software Engineer
-----------------

I’ve been coding for about eight years, but kicked off my professional career in 2021. Since then, I’ve worked on projects across finance, e-commerce, and data dashboards, mostly in full-time roles (with a few fun freelance gigs sprinkled in).

I’m a Fullstack Developer with a degree in Software Engineering — and, honestly, a bit of an architecture addict. I love designing systems that make sense, scale nicely, and don’t turn into spaghetti six months later.

These days I mostly work with Node.js, Go, and Java, usually backed by PostgreSQL or MongoDB, and containerized with Docker on AWS.

### Contact

You can contact me at [schneidtdev@gmail.com](mailto:schneidtdev@gmail.com)

### 💬 A Bit More About My Stack

I’ve worked with most languages out there, but the main ones I’m experienced with are Node.js, PHP, Java, and Go — in that order.

On the frontend side, it’s usually Angular or React for me.

I like to think of myself as a true full-stack developer — I’ve done frontend, backend, infrastructure, mobile, data science, and a lot of academic researching.
The only thing I stay away from is design… because, honestly, that’s just boring 😅

* Building APIs that *just don’t die*
* Clean Architecture, SOLID, and all those fancy acronyms
* TDD/BDD (because broken tests at 2 AM build character)
* Turning complex systems into something readable and maintainable
* Occasionally dabbling in ML projects — especially image classification

### 🌱 Currently

* Probably refactoring something that already worked fine

### 🤝 I’m Open to Collaborating On
